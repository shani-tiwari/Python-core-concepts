# map      
  - used for applying a function to each item in an iterable              
# filter                
  - used for filtering items from an iterable based on a condition              
# reduce                
  - used for reducing an iterable to a single value              
# higher order function 
  - takes another function as an argument or returns a function              
# closure               
  - remembers the environment in which it was created              
# decorator        
  - function that takes another function as an argument and returns a function      
# lambda function      
  - anonymous function
